📌 Fetching Data using Web Scraping

Web scraping is the process of automatically extracting information from websites. Instead of manually copying data, a program (called a scraper) sends requests to a webpage, retrieves the HTML content, and then parses it to collect the required information.

🔑 Key Points
Definition: Automated method to collect data from websites.

How it works:
Send a request to the website (like a browser does).
Download the HTML content of the page.
Parse the HTML to locate specific elements (e.g., product names, prices, headlines).
Store the extracted data in a structured format (CSV, JSON, database).

Tools/Libraries:
Python: BeautifulSoup, Scrapy, Selenium, requests
Other languages also have scraping libraries.
Price monitoring (e.g., e-commerce sites)
Collecting news articles
Gathering research data
Tracking job postings
Limitations:
Some sites block scrapers or require authentication.

Must respect robots.txt and site policies.

Ethical and legal considerations apply.
